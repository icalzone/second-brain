# Weekly Synthesis Checklist (≤30 minutes)

## 1. Ingest (5–10 min)
- [ ] Run VS Code task: Weekly Synthesis
- [ ] Confirm new wiki pages were created/updated
- [ ] Ignore formatting perfection

## 2. Review (10–15 min)
- [ ] Are concepts clearly named?
- [ ] Any duplicate ideas that should merge?
- [ ] Any missing links between related pages?
- [ ] Fix wording that you’d be embarrassed to reread later

## 3. Commit (5 min)
- [ ] One commit per synthesis theme
- [ ] Message answers: *“What did I learn?”*

## 4. Optional Reflection (Bonus)
- [ ] Is anything worth promoting to a project doc?
- [ ] Does this connect to a past failure or win?
