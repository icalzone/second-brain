# Calm Task System (Apple Reminders + Second Brain)

A simple, durable personal productivity system built around Apple Reminders for execution and a lightweight Second Brain for thinking, reference, and learning. The goal: reduce decision fatigue, keep trust in your lists, and recover quickly when everything feels overwhelming.

## Core Rules (Memorize These)

| Rule | Meaning |
|---|---|
| **Dates mean dates** | If something has a date, it *must* matter on that date |
| **Flags mean action** | Flagged = you are intentionally working on it *now* |
| **Backlog is permission to forget** | If it has your attention, it doesn't belong in Backlog |
| **Decisions before actions** | Backlog → Tasks is a decision. Tasks → Flagged is an action |
| **Short lists = calm mind** | Heavy lists are signals, not failures |

## The Mental Model

```
Inbox → Tasks → Flagged (Active)
Inbox → Backlog (no pressure)
```

## Your Lists

| List | Purpose |
|---|---|
| **Inbox** | Capture only — no decisions, no dates |
| **Tasks** | Committed items, not yet active |
| **Backlog** | Ideas, someday, no urgency — permission to ignore |
| **Active** | What you are working on *right now* (≤7 items) |
| **Today** | Due today, fully trusted |
| **Upcoming** | Early warning — items scheduled in the near future |
| **Bills** | Financial tasks with real due dates |

## Daily Rhythm

### Morning (2 minutes)
- Open **Today** and **Active**
- Choose **1–3 outcomes** that would make today successful
- Unflag everything else if needed

### Daily Review (5–10 minutes)
- **Reminders Inbox:** move items to Tasks / Bills / Backlog
- Add dates only if real — never aspirational
- Flag only if actively working
- **Mail intake:** scan for requests or commitments → capture into Reminders Inbox (link back to email if helpful)
- Do not manage tasks from Mail

### Weekly Reset (10–15 minutes)
- Reset Active (≤7 items)
- Scan Upcoming
- Quick Calendar glance (next 7–10 days)
- Promote 1–3 Backlog items
- Check Bills

### Monthly Trust Reset (5–10 minutes)
- Prune Tasks list
- Remove guilt from Backlog
- Push fake dates out or remove them
- Unflag everything, then re-flag only what's genuinely active

## Four Operating Modes

| Mode | Tool | Rule |
|---|---|---|
| **Capture** | Reminders Inbox, Apple Notes, Wallabag, Siri | Fast and dumb — no decisions |
| **Organize** | Reminders (review) | Slow and intentional — once daily or weekly |
| **Plan** | Calendar + Reminders | Lightweight steering, not over-thinking |
| **Execute** | Flagged/Active list | Do the work — no organizing allowed during execution |

## Apple Automation

Shortcuts may: **capture, tag, link, template**  
Shortcuts may not: **auto-prioritize, auto-schedule, move Backlog → Tasks without you**

See [apple-shortcuts-recipes](apple-shortcuts-recipes.md) for concrete recipes.

> If a shortcut makes you feel managed instead of supported, delete it.

## Recovery Protocols

| Problem | Fix |
|---|---|
| Overwhelmed | Unflag everything except 3 |
| Too many flags | You skipped a decision — go through and decide |
| Noisy Today list | Fake dates were added — push them out |
| Heavy Backlog | It's doing its job — that's fine |

## The Second Brain Integration

The task system is for **execution**. The Second Brain (this repo) is for **thinking and reference**. They complement each other:
- Ideas captured in Inbox → if they're learning/reference material, they go into the Second Brain (raw → wiki)
- Tasks reference wiki articles for context (link a reminder to a note)
- Never store reference material in Reminders
